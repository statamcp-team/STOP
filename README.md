# STOP
STOP (StataMCP Team OpenData Projects).

STOP是由上海比翼视界信息科技有限公司与StataMCP-Team共同发起的一个开源数据计划，全称为 StataMCP Team OpenData Projects。希望为中国的社会科学研究者提供开放易用的社科研究数据，并共同推进中国社会科学数据共享与繁荣。
